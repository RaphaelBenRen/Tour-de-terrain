# TabletteUAP

 [Angular CLI](https://github.com/angular/angular-cli) version 15.2.6.

Make sur to run the `npm install` command before everything

## Developpement tool

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Build the apk

Run the command `npm run start` to build Angular and the Cordova application both *(require Gradle and a connexion **(not proxied)**)*
